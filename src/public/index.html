TeamCellMania
API v2.0

Backend multi-tenant para gestión de inventario y facturación electrónica · SRI Ecuador

Servicio activo
Versión 2.0.0
Runtime NestJS + PostgreSQL
Arquitectura Multi-tenant
🔐 La mayoría de endpoints requieren autenticación Bearer JWT. Obtén tu token en POST /auth/login e inclúyelo en el header Authorization: Bearer <token>
🔐
Auth Público

Login, refresh token y gestión de sesión.

POST /auth/login POST /auth/refresh POST /auth/logout
🏢
Empresas SUPERADMIN

Gestión de empresas y suscripciones del sistema.

GET /empresas POST /empresas PUT /empresas/:id
👤
Usuarios Tenant

Usuarios y roles por empresa.

GET /usuarios POST /usuarios PUT /usuarios/:id
🏪
Sucursales Tenant

Puntos de emisión y sucursales por empresa.

GET /sucursales POST /sucursales PUT /sucursales/:id
👥
Clientes Tenant

Clientes con campos adicionales personalizados por factura.

GET /clientes POST /clientes PUT /clientes/:id GET /clientes/:id/campos
📦
Items Tenant

Productos y servicios con control de lotes de inventario.

GET /items POST /items GET /items/:id/lotes POST /items/:id/lotes
📱
Marcas y Modelos Global

Catálogo de dispositivos sincronizado diariamente con GSM Arena.

GET /brands GET /models GET /models/search
🧾
Ventas Tenant

Registro de ventas con detalle por lote y campos adicionales.

GET /ventas POST /ventas GET /ventas/:id
⚡
Facturas Tenant

Facturación electrónica SRI: XML, firma digital y envío SOAP.

POST /facturas GET /facturas/:id POST /facturas/:id/enviar GET /facturas/:id/pdf
🗂️
Catálogos Global

Tablas estáticas del SRI Ecuador. Solo lectura.

GET /catalogos/formas-pago GET /catalogos/tarifas GET /catalogos/comprobantes
✍️
Firmas Tenant

Certificados digitales p12 para firma electrónica.

GET /firmas POST /firmas/upload PUT /firmas/:id/activar
📋
Suscripciones SUPERADMIN

Planes, suscripciones y control de vencimientos.

GET /planes GET /suscripciones POST /suscripciones
